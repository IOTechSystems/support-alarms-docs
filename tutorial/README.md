# Alarms Service Tutorial

## Prerequisites

- Edgecentral >= v4
- jq
- curl

## Running

To run the demo

```bash
./deploy.sh
```

Once the demo is running, to Add the devices and profiles:

```bash
./add_devices.sh
```


To configure Conditions, Routes, and Actions please follow steps [here](./docs/contents.md).



To stop the tutorial

```bash
edgecentral down
```
